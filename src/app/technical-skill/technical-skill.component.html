Description

React est une bibliothèque JavaScript développée par Facebook depuis 2013. Cette bibliothèque est utilisée pour développer la partie frontend d'une application web grâce à son fonctionnement basé sur le développement de composants d'interface utilisateur réutilisables. Cette approche permet de réduire la charge de travail du développeur en réutilisant les petits bouts d'interface.
