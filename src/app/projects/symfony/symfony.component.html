Symfony Project

Projet personnel de gestion de données avec Symfony

Présentation et Contexte

Projet personnel de développement d'une application CRUD avec Symfony. L'application permet de gérer des données de manière efficace et sécurisée.

Objectifs

  • Développement d'une application CRUD
  • Gestion des données
  • Interface utilisateur intuitive
  • Architecture robuste
