Portfolio

Développement d'un site web personnel

Présentation et Contexte

Ce portfolio est un projet personnel visant à présenter mes compétences et réalisations professionnelles. Il a été développé pour démontrer mes capacités en développement web et en design.

Objectifs

  • Développer une application web moderne et responsive
  • Présenter mes compétences et projets de manière professionnelle
  • Appliquer les meilleures pratiques de développement
  • Créer une interface utilisateur intuitive et esthétique

  • Conception de l'architecture
  • Choix des technologies

Phase 2 : Développement

  • Configuration de l'environnement
  • Développement du backend
  • Création de l'interface utilisateur

Phase 3 : Tests et Déploiement
